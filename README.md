# MedManager-V1-JAVA
Gestion hospitalière v1.0 - Architecture Orientée Objet (Java 25)

## 🧬 Évolution POO
ce projet utilise désormais une classe `Patient` pour modéliser les données :
- **Encapsulation** : Les données du patient (Nom, Prénom, Age, Service) sont regroupées dans un seul objet.
- **Logique métier** : Les calculs (comme l'âge) sont gérés directement par la classe `Patient`.
- **Simplification** : Le code de tri et de recherche est plus robuste et facile à maintenir.

- ## 🚀 Installation
1. Téléchargez le fichier `.exe` dans la section [Releases].
2. Lancez l'application sous Windows.
